# Desktop door images

Place the following files here (exact filenames, case-sensitive):

- `Left-door-Dekstop.webp`   ← referenced in code as `/desktop/Left-door-Dekstop.webp`
- `Right-Door-Dekstop.webp`  ← referenced in code as `/desktop/Right-Door-Dekstop.webp`

These files are served statically by Vite from the `public/` folder.
They must be present before running `npm run dev` or `npm run build`.
